A Math-Defense Game

MathStrike

Defend Earth by solving math problems. Every shot is a math problem solved.

MathStrike · v1.0

How to Play

1
Meteors fall toward Earth, each with a shape and a math equation.
2
Tap (or use keys) the matching shape AND the correct answer (a single digit, 0–9).
3
A missile fires. If both inputs match a meteor, it gets destroyed.
4
Don't let meteors hit Earth. 0% health = game over.
★
Destroy 2 meteors within half a second for a double-kill bonus. Fast meteors are worth more.
KEYBOARD CONTROLS
Q W A S = Triangle / Circle / Square / Diamond
0–9 = Answer digits
Esc / P = Pause
